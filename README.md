# Dizi Codex Pet

Dizi（谛仔）是一只友好机灵的黑白 Codex 动画宠物，拥有大号黑色三角耳、白色脸部和胸前的三道蓝色短线。

![Dizi animation contact sheet](preview/contact-sheet.png)

## 安装方法

1. 下载本仓库中的 `pet.json` 和 `spritesheet.webp`。
2. 打开 Codex 的 **Settings → Pets**。
3. 选择 **Import Codex sprite**。
4. 导入 `spritesheet.webp`，并按界面提示完成设置。

也可以在仓库右侧的 **Releases** 中下载完整 ZIP 安装包。

## 文件说明

- `pet.json`：宠物名称、描述和图集路径。
- `spritesheet.webp`：Codex 兼容的透明动画图集。
- `preview/contact-sheet.png`：全部动画状态的预览图。

## 图集规格

- 尺寸：1536 × 1872
- 网格：8 列 × 9 行
- 单元格：192 × 208
- 格式：RGBA WebP
- 动画：idle、running-right、running-left、waving、jumping、failed、waiting、running、review

## English

Dizi is a friendly black-and-white animated pet for Codex, with large black ears and three blue marks on its chest.

Download `pet.json` and `spritesheet.webp`, then import the sprite from **Codex Settings → Pets → Import Codex sprite**.
